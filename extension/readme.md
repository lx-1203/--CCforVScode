# 星迹的CC✨

> **版本说明**：本扩展采用独立版本号体系。`0.x.x` 为星迹的CC 自有版本，基于官方 Claude Code `2.1.95` 汉化增强。

在你的编辑器中释放 Claude 的强大能力。即时搜索百万行代码库，将数小时的工作流程缩短为一条指令。你的工具、你的工作流、你的代码库，以思维的速度进化。

- **强大的智能：** 使用最新的 Claude 模型，支持 Pro、Max、Team 或 Enterprise 订阅，也支持按用量计费
- **与你协作：** Claude 自主探索你的代码库，读写代码，并在你的许可下运行终端命令
- **全新友好界面：** 更易上手的交互体验
- **与编辑器深度集成：** Claude 了解你当前的文件和选中的文本，直接在编辑器窗口中提出修改建议
- **强大的 Agent 功能：** 支持子代理、自定义斜杠命令和 MCP（这些功能在 VS Code 扩展中可用，但部分功能需要通过命令行界面配置）
- **🔊 语音朗读（v0.1.3 新增）：** Claude 回复可自动朗读，使用系统内置 TTS，支持中文

## 版本历史

| 星迹版本 | 对应官方版本 | 主要变更 |
|---------|------------|---------|
| 0.1.3 | 2.1.95 | 新增语音朗读（TTS）、启用语音输入（STT） |
| 0.1.2 | 2.1.95 | Pre-Send 预发送队列、音频通知增强 |
| 0.1.1 | 2.1.93 | 初始汉化增强版 |

## ✨ 特色功能

- **YOLO 模式：** 跳过所有权限确认，让 Claude 全自动执行所有操作
- **全面汉化：** 界面和提示信息已翻译为中文
- **自定义 API 配置：** 支持配置自定义 API 密钥和基础 URL

## 刚接触 Claude Code？

访问 [claude.com/claude-code](https://claude.com/claude-code) 开始使用。

## 更喜欢终端模式？

如果你想要之前的终端风格体验，使用 `星迹的CC✨: 在终端中打开` 设置即可切换回去。

## 系统要求

- VS Code 1.98.0 或更高版本
